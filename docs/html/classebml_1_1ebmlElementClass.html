Detailed Description

Abstract base class for EBML Element Type objects. Every subclass of ebmlElementClass must have a companion subclass of ebmlElement declared as a friend class.

Instances of ebmlElementClass are constructed using an EBML ID, either as an ebmlID_t (unsigned long long) or as a VINT (const char*), and a class name (unicode string).
